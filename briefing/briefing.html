Function Explanation

18/8/2022 - DOM (Document Object Model)
DOM use for:
1 Change the content of HTML elements
2 Change the style (CSS) of HTML elements
3 React to HTML DOM events
4 Add and delete HTML elements

0. How to target HTML elements

Example:
document.getElementById("example") Target element with Id = "example" Link
document.getElementByClassName("example") Target ALL element with Class = "example" Link
document.getElementsByTagName("h1") Target ALL element by Tagname e.g (h1, p, etc.. ) Link
document.querySelectorAll("p.intro") Target ALL element by Tagname WITH/WITHOUT certain id or class Link
Others DOMs

Example set DOM as object:

const example = document.getElementById("example")

const example = document.getElementByClassName("example")

const x = document.getElementsByTagName("p")

const y = document.querySelectorAll("p.intro");


1. Change the content of HTML elements

code Explaination
.innerHTML The easiest way to modify the content of an HTML element Link
.attribute Changes the HTML element attribute Link
.write Used to write directly to the HTML Link

Example:

document.getElementById("example").innerHTML = "Hello World!"

2. Change the style (CSS) of HTML elements

Change elements style
